System: Mac (Xcode)

Path: Using relative path to read the file, please put the coding file and reading file under the same path


Compile: using command 'make' to run Makefile. This will produce 'output'.

Run: 
1) insert sort method: using the command './output' to run the output. This would be the default argument.
2) heap sort method: using the command './output 1' to run the output. 
3) quick sort method: using the command './output 2' to run the output.
